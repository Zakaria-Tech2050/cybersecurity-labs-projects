# Multi-OS Environment Lab

## Overview
Designed and deployed a virtualized lab environment using multiple operating systems for system administration practice and testing.

## Environment
- Windows Server 2022 (AD, DNS, server administration)
- Windows 10 (domain-joined client)
- Ubuntu Server (Linux services & CLI administration)
- Ubuntu Desktop (GUI-based Linux usage)

## Tools Used
- VMware Workstation Player
- VirtualBox

## Skills Gained
- Virtualization
- OS installation & configuration
- Active Directory fundamentals
- Linux system administration
